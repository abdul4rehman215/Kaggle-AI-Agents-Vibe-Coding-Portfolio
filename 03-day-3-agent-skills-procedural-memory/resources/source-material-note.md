# 📚 Source Material Note - Day 3

This file documents what material was used for Day 3 theory notes and what was intentionally not committed to the repository.

---

## Source material used

The Day 3 notes were based on:

- Unit 3 podcast/video summary
- Agent Skills whitepaper
- NoteGPT-style podcast summary
- NotebookLM study guide
- NotebookLM quiz / Q&A review
- NotebookLM explanation video
- Personal visual summaries created during revision

---

## What is committed

This folder includes:

- human-written study notes,
- a compact glossary,
- a study-process summary,
- official links,
- two visual study assets.

The goal is to document what I understood, not to archive every generated note.

---

## What is not committed by default

The following are intentionally not added to the repo at this stage:

- the full whitepaper PDF,
- raw NoteGPT output,
- raw NotebookLM generated notes,
- private account screenshots,
- local terminal history,
- API keys,
- credentials,
- local config files,
- or anything that could expose a private environment.

The official whitepaper is linked in `day-3-links.md` instead of being duplicated here.

---

## Why this matters

Good course documentation should be useful without becoming noisy.

For this folder, I only kept the parts that help me revise and explain the concept later:

- what Agent Skills are,
- how progressive disclosure works,
- why context rot matters,
- how Skills compare with MCP and `AGENTS.md`,
- why evaluation is part of the skill lifecycle,
- and what needs to be validated during the codelabs.

This keeps the folder clean now and leaves space for real implementation notes after the hands-on work is completed.
