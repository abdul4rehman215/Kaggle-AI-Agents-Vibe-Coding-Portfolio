# Weather Assistant Minimal Snapshot

This folder keeps a small, reviewable snapshot of the prototype generated during the Day 3 Antigravity Skills codelab.

The full generated environment was intentionally not committed. Virtual environments, local ADK session databases, cache folders, and run logs were removed. The prototype was used to confirm Agents CLI scaffolding, dependency sync, ADK playground startup, and graph rendering. Live model calls through Vertex AI were blocked by Cloud billing on the active project, so the limitation is documented in the codelab notes instead of hiding the failure.
