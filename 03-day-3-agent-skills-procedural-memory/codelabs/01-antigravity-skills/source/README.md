# Source Snapshot - Codelab 1 Antigravity Skills

This folder keeps the small source artifacts worth preserving from the Antigravity Skills codelab.

| Folder | Purpose |
|---|---|
| [`workspace-skills/`](./workspace-skills/) | Curated copies of the workspace skills used during the codelab. |
| [`skill-test-artifacts/`](./skill-test-artifacts/) | Small files created to test the skills. |
| [`weather-assistant-minimal/`](./weather-assistant-minimal/) | Minimal scaffold snapshot from the Agents CLI weather assistant prototype. |

Local runtime folders, virtual environments, caches, session databases, and credentials were intentionally excluded.
