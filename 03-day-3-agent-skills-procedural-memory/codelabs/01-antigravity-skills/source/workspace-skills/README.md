# Workspace Skills Snapshot

This folder keeps reviewable copies of the skills used in Codelab 1.

The original workspace skill path during the lab was:

```text
my-first-project/.agents/skills/
```

The uploaded workspace ZIP did not include the hidden `.agents/skills/` folder directly, so this snapshot preserves a clean, documented version of the same codelab skill set based on the inspected `SKILL.md` content and observed behavior.

| Skill | Role |
|---|---|
| `code-review` | Baseline project skill already present before the codelab. |
| `git-commit-formatter` | Formats Git commit messages using Conventional Commits. |
| `license-header-adder` | Adds a standard license header to new source files. |
| `json-to-pydantic` | Converts JSON examples into Pydantic models. |
| `database-schema-validator` | Runs a script-backed SQL schema policy check. |

This folder is included for learning/review, not as a package manager replacement.
