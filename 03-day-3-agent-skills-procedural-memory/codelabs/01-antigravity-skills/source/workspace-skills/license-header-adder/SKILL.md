# License Header Adder Skill

This skill ensures that new source files include the standard open-source license header.

## Instructions

1. Read `resources/HEADER_TEMPLATE.txt`.
2. Prepend the header to new source files.
3. Convert the comment syntax for Python, shell, YAML, SQL, or C-style languages as needed.
4. Do not duplicate the header if the file already contains it.
