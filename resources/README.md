# 📚 Resources

This folder keeps official course links and useful references separate from daily implementation notes.

The goal is to avoid scattering important links across multiple README files. Day-wise folders can still link to the specific resources they use, but this folder works as the central reference point.

---

## 📌 Files

| File | Purpose |
|------|---------|
| [`official-links.md`](./official-links.md) | Main official links for the course, tools, codelabs, and capstone |
| [`official-course-links.md`](./official-course-links.md) | Course-specific Kaggle and Google links grouped with short notes |

---

## 🧭 How I Use This Folder

I use this folder when I need to quickly reopen:

- the Kaggle course page
- the daily codelabs
- the capstone competition
- Google AI Studio or Cloud Run references
- official material that should be cited or revisited later

This folder should stay focused on references. Actual notes, implementation details, screenshots, and reflections belong inside the relevant day-wise folders.
