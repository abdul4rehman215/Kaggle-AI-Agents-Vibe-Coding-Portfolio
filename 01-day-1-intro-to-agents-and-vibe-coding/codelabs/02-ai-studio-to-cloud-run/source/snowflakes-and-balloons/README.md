# ❄️🎈 Snowflakes & Balloons

A small browser-only React/Vite app built during the Day 1 AI Studio codelab.

## What it does

The app lets the user trigger falling snowflakes and floating balloons from explicit button clicks. It includes active counters, 5-second progress indicators, optional Web Audio sound, reduced-motion controls, and a clean reset action.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Privacy / dependency notes

This portfolio copy does not require login, a backend, a database, analytics, paid APIs, or a Gemini API key. The visual effects run in the browser.

The app was deployed and tested through Cloud Run during the codelab, then unpublished to avoid ongoing cost.
